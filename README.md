2.Individual Approach

驱动方式
我选择使用柏林噪声和随机性作为我的个人作业的动画化方法，主要通过柏林噪声和随机性控制画面中形状的颜色的变化和形状振动的频率。这种方式能够更好的呈现蒙德里安的Broadway Boogie Woogie中线条和方块富有韵律的排布方式，使整个画面更加生动有趣

动画化的属性
A模式中条带根据

与其他人的区别：我的原型侧重于通过柏林噪声值的变化，不仅驱动几何图形的颤动，同时控制颜色的变化；并在B模式中加入会产生变化的小猫头像，增加了画面的趣味性
